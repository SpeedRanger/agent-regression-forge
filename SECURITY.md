# Security And Public Safety

Agent Regression Forge is designed for public or redacted examples.

Do not put any of these in the app, public issues, launch comments, or examples:

- secrets
- tokens
- API keys
- passwords
- private repository contents
- private logs
- private analytics
- customer data
- payment details
- NDA material

If a request needs private material, do not post it publicly. Use a
buyer-approved private channel after scope-fit confirmation.

## Reporting a concern

Open a public issue only when the report can be fully described with public or
redacted information. Otherwise, use a private channel controlled by the repo
owner.
