# Reserve A Regression-Pack Slot

Agent Regression Forge has five launch-sprint slots for a `$19`
public/redacted failure-to-regression-pack service.

## Slot Rules

- One public/redacted failed agent run per slot.
- No private repos, private logs, secrets, tokens, customer data, or payment
  details in public issues.
- Delivery target is within 24 hours after scope-fit confirmation and payment
  evidence.
- Payment counts only with receipt, checkout export, paid invoice, or equivalent
  buyer-confirmed evidence.

## Request Path

Open the public request form:

https://github.com/SpeedRanger/agent-regression-forge/issues/new?template=regression_pack_request.yml

## What To Paste

- Intended behavior
- Stack or test runner
- Public/redacted agent trace
- Public/redacted observed breakage
- Generated app output, if available

## What Not To Paste

- secrets
- tokens
- API keys
- passwords
- private repo contents
- private logs
- customer data
- payment details
- NDA material
